# CMSC 207 Mini-Project: Personal Web Profile

## Project Overview
This is a single-page personal web profile built for my CMSC 207 class. It serves as a foundational project to apply core web development concepts, including semantic structure, external styling, and basic DOM manipulation. 

## Technologies Used
* **HTML5:** Used proper semantic tags to structure the content.
* **CSS3:** Built an external stylesheet for a custom dark-mode design, layout management, and responsive media queries.
* **Vanilla JavaScript:** Added simple client-side interactivity without relying on external libraries.

## Key Features
* **Responsive Layout:** The design adapts seamlessly from desktop monitors down to mobile screens.
* **Interactive UI:** Includes CSS hover animations on navigation links, interactive buttons, and content cards.
* **Dynamic Content:** Features a JavaScript-powered button that reveals a hidden fun fact when clicked.
